About This Map

A weighted overlay model was constructed using four input rasters: land cover class (weight 35%), canopy cover percentage (weight 25%), distance to perennial streams (weight 25%), and slope class (weight 15%). Each raster was reclassified to a 1–9 suitability scale prior to overlay. The resulting habitat suitability index was validated against known occurrence records and used to identify priority conservation blocks.
