About This Map

This hillshade layer was derived from 1-meter bare-earth LiDAR data acquired through the USGS 3D Elevation Program (3DEP). The raster was processed to remove vegetation and structures, isolating ground returns to produce a clean terrain surface.

The hillshade is rendered at 315° azimuth and 45° altitude with a z-factor appropriate to the data coordinate system. It was used as a base layer for identifying micro-topographic features including channels, ridgelines, landslide scarps, and legacy road drainage structures.
